# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Community health files (CONTRIBUTING.md, CODE_OF_CONDUCT.md, SECURITY.md)
- Issue and PR templates
- GitHub Actions CI workflow
- Social preview image

## [5.0.0] - 2026-01-19

### Added
- OMNI-PRIME v5.0 psychological analysis engine
- PsychoCore-X and PsychoCore-ULTRA analysis systems
- Big Five (OCEAN) personality profiling
- MBTI personality type detection
- Enneagram analysis
- Jung Archetypes integration
- BART (Balloon Analogue Risk Task) risk tolerance test
- Multi-cultural support (Western, Eastern, African frameworks)
- Personalized content curation (movies, books, music)
- Cloudflare Workers deployment
- Ollama and Google Gemini AI integration
- Modern TypeScript + Hono stack
- Vite build system

### Changed
- Migrated to Cloudflare Workers for edge computing
- Improved cultural calibration algorithms
- Enhanced personality analysis accuracy

### Security
- Implemented secure API key management
- Added data encryption for user profiles
- GDPR compliance measures

## [4.0.0] - 2025-12-01

### Added
- Cultural context awareness
- Multi-language support
- Enhanced BART test

### Changed
- Refactored analysis engine
- Improved UI/UX

## [3.0.0] - 2025-09-15

### Added
- MBTI and Enneagram support
- Content recommendation system

## [2.0.0] - 2025-06-01

### Added
- Big Five personality analysis
- Basic BART test

## [1.0.0] - 2025-03-01

### Added

- Basic psychological profiling
- Simple content curation
